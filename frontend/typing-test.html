Console Instructions

To test in your app, open the browser console and run:

// Test showing typing indicator
app.handleTypingIndicator({
    user_id: 123,
    username: 'Test User',
    is_typing: true,
    group_id: null  // or set to group ID if testing group chat
});

// Test hiding typing indicator
app.handleTypingIndicator({
    user_id: 123,
    username: 'Test User',
    is_typing: false,
    group_id: null
});
